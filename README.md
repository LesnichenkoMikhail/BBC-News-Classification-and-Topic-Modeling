# Классификация новостей BBC и тематическое моделирование 
Описание проекта

Данный проект посвящён задаче обработки текстовых данных новостей BBC.
Основная цель — построить модель, которая автоматически классифицирует новости по тематическим категориям, а также дополнительно исследовать скрытую тематическую структуру текста с помощью методов машинного обучения и NLP.

Проект реализует полный цикл работы с текстовыми данными: от загрузки и очистки данных до построения моделей, оценки качества и анализа результатов.

Задачи проекта

В рамках проекта решались следующие задачи:

Классификация новостей на 5 тематических категорий:
business
sport
politics
tech
entertainment
Построение базовой NLP-системы для обработки текстов
Анализ распределения классов и структуры данных
Выявление скрытых тем в корпусе текстов с помощью методов тематического моделирования
Данные

Используется открытый датасет BBC News из HuggingFace:

1225 текстов в обучающей выборке
1000 текстов в тестовой выборке
5 сбалансированных классов новостей

Каждый объект содержит:

текст новости
числовую метку класса
текстовую метку категории
Этапы работы
1. Загрузка и первичный анализ данных
Датасет загружен с использованием библиотеки datasets
Проведён анализ структуры данных
Изучено распределение классов
Проверена сбалансированность выборки
2. Предобработка текста

Для подготовки текста к обучению использовалась библиотека spaCy.

Были выполнены следующие шаги:

приведение текста к нижнему регистру
удаление цифр и пунктуации
токенизация
лемматизация слов
удаление стоп-слов
фильтрация коротких и неинформативных токенов

Результатом стал очищенный текст, готовый для векторизации.

3. Преобразование текста в признаки

Для преобразования текста в числовое представление использовался метод TF-IDF:

учитывались униграммы и биграммы (n-gram range = 1–2)
ограничение размера словаря до 5000 признаков
фильтрация редких слов (min_df = 2)

Это позволило выделить наиболее значимые слова и устойчивые фразы для каждой категории.

4. Обучение моделей машинного обучения

Для задачи классификации были обучены две классические модели:

Logistic Regression
Linear Support Vector Machine (Linear SVM)

Обе модели обучались на TF-IDF признаках.

5. Оценка качества моделей

Для оценки использовались стандартные метрики:

Accuracy
Precision
Recall
F1-score
Confusion matrix

Результаты:

Logistic Regression: Accuracy ≈ 96.6%
Linear SVM: Accuracy ≈ 96.9%

Модели показали высокое качество на тестовой выборке.

6. Анализ результатов и интерпретация

Дополнительно проведён анализ интерпретируемости моделей:

извлечены наиболее важные слова для каждого класса
проанализированы различия между тематическими категориями
построены confusion matrix для оценки ошибок классификации

Это позволило понять, какие признаки влияют на предсказания модели.

7. Тематическое моделирование (BERTopic)

Для более глубокого анализа текстов был использован BERTopic:

SentenceTransformers (all-MiniLM-L6-v2) для получения эмбеддингов
CountVectorizer для построения представления текста
BERTopic для выделения скрытых тем

Также использовалась визуализация:

UMAP для снижения размерности
графическое отображение тематических кластеров
анализ распределения документов по темам

Это позволило выявить скрытую структуру текстового корпуса без использования разметки.

Основные результаты

В ходе работы были получены следующие результаты:

построена end-to-end система обработки текста
достигнута высокая точность классификации (до 97%)
получены интерпретируемые признаки для каждого класса
выделены тематические кластеры в тексте с помощью BERTopic
визуализирована структура текстовых данных
