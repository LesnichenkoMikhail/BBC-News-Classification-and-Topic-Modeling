# Классификация новостей BBC и тематическое моделирование 

## Описание проекта

Данный проект посвящён обработке текстов новостей BBC с использованием методов обработки естественного языка (NLP).  
Основная цель — построить модель, которая автоматически классифицирует новости по тематическим категориям и дополнительно анализирует скрытую тематическую структуру текста.

Проект реализует полный NLP pipeline: от загрузки данных до обучения моделей, оценки качества и тематического моделирования.



## Цели проекта

- Классификация новостей на 5 категорий:
  - business
  - sport
  - politics
  - tech
  - entertainment

- Построение полного пайплайна обработки текста  
- Анализ структуры данных и распределения классов  
- Выявление скрытых тематик в текстах  



## Данные

Используется датасет **BBC News** из HuggingFace:

- 1225 объектов в обучающей выборке  
- 1000 объектов в тестовой выборке  
- 5 сбалансированных классов  

Каждый объект содержит:
- текст новости  
- числовую метку класса  
- текстовую метку категории  



## Этапы работы

### 1. Загрузка и анализ данных
- загрузка датасета через `datasets`
- анализ структуры данных
- изучение распределения классов
- проверка баланса выборки

### 2. Предобработка текста
Использовалась библиотека **spaCy**:

- приведение к нижнему регистру  
- удаление цифр и пунктуации  
- токенизация  
- лемматизация  
- удаление стоп-слов  
- фильтрация коротких токенов  

### 3. Векторизация текста
Использовался **TF-IDF**:

- n-gram range: (1, 2)  
- max_features = 5000  
- min_df = 2  

### 4. Обучение моделей
Обучены две модели:

- Logistic Regression  
- Linear SVM  

### 5. Оценка качества

Метрики:
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion matrix  

Результаты:
- Logistic Regression: ~96.6%  
- Linear SVM: ~96.9%  

### 6. Интерпретация моделей
- анализ важных признаков для каждого класса  
- анализ ошибок (confusion matrix)  
- понимание различий между категориями  

### 7. Тематическое моделирование (BERTopic)

Использовались:

- SentenceTransformers (all-MiniLM-L6-v2)  
- CountVectorizer  
- BERTopic  

Дополнительно:
- UMAP для визуализации  
- анализ тематических кластеров  



## Результаты

- построен полный NLP pipeline  
- достигнута точность до 97%  
- получены интерпретируемые признаки классов  
- выделены тематические кластеры  
- выполнена визуализация структуры текста
